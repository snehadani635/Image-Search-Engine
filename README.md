# 🖼️ Image Search Engine

A simple and responsive Image Search Engine built using **HTML, CSS, and JavaScript** that allows users to search for images using the **Unsplash API**.

## 🚀 Features

- Search images by keyword
- Fetch real-time images from Unsplash API
- Responsive user interface
- Show More functionality for additional results
- Clean and modern design


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Unsplash API

## ⚙️ How It Works

1. User enters a search term.
2. The application sends a request to the Unsplash API.
3. Relevant images are fetched and displayed.
4. Clicking the **Show More** button loads additional images.
